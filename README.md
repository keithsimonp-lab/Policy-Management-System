## 📌 Project Description
This project is a simple Insurance Policy Management System developed in Python.  
It allows an insurance company to manage policyholders, products, and payments.

The system demonstrates object-oriented programming concepts such as classes, methods, and file separation.

---

## ✅ Features

### 1. Policyholder Management
- Register policyholders
- Suspend policyholders
- Reactivate policyholders
- View policyholder account details

### 2. Product Management
- Create new insurance products
- Update product details
- Suspend or activate products

### 3. Payment Management
- Process payments
- Send payment reminders
- Apply penalties to late payments

## 💻 System Demonstration

The system performs the following actions:

1. Creates an insurance product (Health Insurance)
2. Creates two policyholders
3. Registers both policyholders to the product
4. Processes payments for both policyholders
5. Allows user to:
   - Suspend a policyholder
   - Reactivate a policyholder
6. Displays account details for both policyholders
